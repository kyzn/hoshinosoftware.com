Code Reader – Support

Code Reader scans barcodes and QR codes using your iPhone's camera or any image in your photo library. It works entirely offline, stores nothing in the cloud, and never shows ads.

<screenshot> <screenshot> <screenshot> <screenshot>

Camera

The Camera tab opens immediately to a live viewfinder. Point it at any barcode or QR code — it detects codes automatically and shows the result.

<screenshot>
  • Flashlight — tap the flashlight button to illuminate dark environments. Automatically disabled when using the front camera.
  • Front camera — tap the rotate button to switch between rear and front cameras. Useful for scanning codes on another screen.
💡 The app shows a red scanning line while it looks for codes. You don't need to press anything — recognition is automatic.

If the camera is not accessible

If Code Reader doesn't have camera permission, a message will appear with an Open Settings button. Tap it, then enable Camera access for Code Reader.

Gallery

The Gallery tab lets you scan codes from an existing image. Tap Choose Photo to pick any image from your library — only the specific photo you choose is shared with the app. Code Reader does not request access to your entire photo library.

<screenshot>

After picking a photo, the app scans it and shows the result immediately. If no code is found, an alert will tell you so.

Scan Results

When a code is detected, Code Reader shows you its content. For plain text or URLs, you'll see the raw data and buttons to Copy, Share, and Open (URLs only).

Tracking parameter detection

When a URL contains tracking parameters (UTM tags, click IDs, etc.), Code Reader shows an Open Without Tracking button. Tap it to open the link with all tracking parameters stripped — you reach the same page without being tracked.

<screenshot>

History

Every scan is automatically saved to the History tab. Scans are kept for 6 months and the history holds up to 1,000 entries; the oldest are removed automatically when the limit is reached.

<screenshot>
  • View a past scan — tap any row to re-open the full result.
  • Delete one entry — swipe left on a row and tap Delete.
  • Delete all — tap the trash icon in the top-right corner.
  • Source label — each entry shows whether it came from the Camera, Gallery, or a Share.

Settings

<screenshot>

Multi-Code Mode

By default, Code Reader stops as soon as it finds one code. Turn on Multi-Code Mode to scan up to 5 codes at once — useful for business cards, product sheets, or any image containing several codes side by side.

When multi-code mode is active, the camera keeps scanning for a brief moment after the first detection to gather all visible codes before showing results. This is why it takes slightly longer than single-code mode.

<screenshot>

Show Details (Advanced Mode)

Turn on Show Details to reveal technical metadata about each code — things like QR version, error correction level, mask pattern, or barcode structure (start/stop characters, check digit). This information is shown both in the result view and in the History tab rows.

<screenshot>

Scanning from Another App

You can scan a code that appears in a photo inside any other app — a message, email, document viewer, or web browser — without leaving that app.

Using the Share Sheet (Share Extension)

  1. Long-press the image that contains the code.
  2. Tap Share.
  3. In the share sheet, find and tap Code Reader.
  4. The app opens and scans the image automatically.
<screenshot>

Using the Actions Row (Action Extension)

  1. Long-press the image that contains the code.
  2. Tap Share.
  3. Scroll the actions row and tap Scan for Codes.
  4. The app opens and scans the image automatically.
<screenshot>
💡 Both options do the same thing. "Code Reader" in the app row and "Scan for Codes" in the actions row are two ways to reach the same feature.

Homescreen Widget

Code Reader includes a homescreen widget in three sizes (small, medium, large). Tapping it opens the app directly to the camera, ready to scan.

<screenshot>

To add the widget:

  1. Long-press an empty area of your homescreen until the icons jiggle.
  2. Tap the + button in the top-left corner.
  3. Search for Code Reader.
  4. Choose a size and tap Add Widget.

Control Center

A Control Center button lets you open Code Reader's camera with a single tap from anywhere — lock screen, another app, anywhere.

<screenshot>

To add the control:

  1. Open Settings → Control Center.
  2. Scroll down to More Controls.
  3. Tap + next to Code Reader.
💡 On iOS 26, you can also add controls directly from the Control Center: swipe to open it, tap +, and find Code Reader in the list.

FAQ

  • Does it need an internet connection?
    No. Code Reader works completely offline.

  • Does it collect my data?
    No. Nothing is uploaded anywhere. Scan history is stored only on your device.

  • Does it access my entire photo library?
    No. In the Gallery tab, you pick a single photo using the system photo picker — Code Reader never receives access to your library.

  • Why does multi-code mode take a little longer?
    The camera gathers codes across a short window of frames to make sure it catches all codes in view before showing the result.

  • How long is scan history kept?
    Entries older than 6 months are removed automatically. The history also caps at 1,000 entries; the oldest are removed to make room for new ones.

  • Why doesn't it navigate to a link automatically?
    This is intentional. Fake or malicious QR codes are common — in parking lots, restaurant tables, and elsewhere. Showing you the URL before opening it lets you verify it's safe.
