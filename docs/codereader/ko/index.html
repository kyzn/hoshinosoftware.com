코드 리더

코드 리더는 iPhone 카메라 또는 사진 보관함의 이미지를 사용하여 바코드와 QR 코드를 스캔합니다. 완전히 오프라인으로 작동하며, 클라우드에 아무것도 저장하지 않고 광고를 표시하지 않습니다.

문의 전 확인 사항

자주 묻는 질문은 아래 설명서와 FAQ에서 확인하실 수 있습니다. 빠른 점검 사항:

  • 카메라가 작동하지 않나요? iPhone 설정 → 개인 정보 보호 및 보안 → 카메라 → 코드 리더 접근 권한을 활성화하세요.
  • 히스토리를 찾을 수 없나요? 히스토리 탭을 누르세요. 항목은 6개월간 보관되며 최대 1,000개까지 저장됩니다.
  • 코드가 스캔되지 않나요? 조명을 밝게 하고 코드를 흔들리지 않게 고정하세요. 이미지 파일이 있다면 갤러리 모드를 사용해 보세요.
  • 앱이 충돌하나요? 강제 종료 후 다시 실행하세요. 문제가 지속되면 iPhone 모델과 iOS 버전을 알려주시고 이메일로 문의해 주세요.

자주 묻는 질문

  • 인터넷 연결이 필요한가요?
    아니요. 코드 리더는 완전히 오프라인으로 작동합니다.

  • 내 데이터를 수집하나요?
    아니요. 어디에도 업로드되지 않습니다. 스캔 기록은 기기에만 저장됩니다.

  • 전체 사진 보관함에 접근하나요?
    아니요. 갤러리 탭에서는 시스템 사진 선택기를 사용하여 사진 한 장만 선택합니다. 코드 리더는 보관함에 대한 접근 권한을 받지 않습니다.

  • 다중 코드 모드가 조금 더 오래 걸리는 이유는 무엇인가요?
    카메라는 결과를 표시하기 전에 시야 내의 모든 코드를 캡처하기 위해 짧은 프레임 창에 걸쳐 코드를 수집합니다.

  • 스캔 기록은 얼마나 오래 보관되나요?
    6개월이 지난 항목은 자동으로 삭제됩니다. 기록은 1,000개 항목으로도 제한되며, 새 항목을 위한 공간을 만들기 위해 가장 오래된 항목이 삭제됩니다.

  • 링크로 자동으로 이동하지 않는 이유는 무엇인가요?
    이는 의도적인 것입니다. 가짜 또는 악의적인 QR 코드는 주차장, 식당 테이블 등 어디서나 흔히 발견됩니다. 열기 전에 URL을 보여주면 안전한지 확인할 수 있습니다.

설명서

카메라

카메라 탭이 열리면 바로 스캔을 시작합니다. 바코드나 QR 코드에 폰을 갖다 대면 앱이 자동으로 감지합니다. 카메라 권한이 필요합니다. 아직 허용하지 않았다면 요청 창이 나타납니다. 설정 열기를 탭하고 코드 리더의 카메라 접근을 활성화하세요.

  • 손전등: 어두운 환경을 밝히려면 손전등 버튼을 탭하세요. 후면 카메라에서만 사용 가능합니다.
  • 전면 카메라: 후면과 전면 카메라를 전환하려면 회전 버튼을 탭하세요. 후면 카메라를 사용할 수 없을 때(예: 파손된 경우) 유용합니다.
💡 앱은 코드를 찾는 동안 빨간 스캔 선을 표시합니다. 아무것도 누를 필요가 없습니다. 인식은 자동으로 이루어집니다.

갤러리

갤러리 탭에서는 기존 이미지에서 코드를 스캔할 수 있습니다. 사진 선택을 탭하여 보관함에서 이미지를 선택하세요. 선택한 특정 사진만 앱과 공유됩니다. 코드 리더는 전체 사진 보관함에 대한 접근을 요청하지 않습니다.

사진을 선택하면 앱이 즉시 스캔하고 결과를 표시합니다. 코드가 발견되지 않으면 알림이 표시됩니다.

기록

모든 스캔은 기록 탭에 자동으로 저장됩니다. 스캔은 6개월 동안 보관되며 기록에는 최대 1,000개의 항목이 저장됩니다. 한도에 도달하면 가장 오래된 항목이 자동으로 삭제됩니다.

  • 이전 스캔 보기: 전체 결과를 다시 열려면 행을 탭하세요.
  • 항목 하나 삭제: 행을 왼쪽으로 밀고 삭제를 탭하세요.
  • 전체 삭제: 오른쪽 상단의 휴지통 아이콘을 탭하세요.
  • 출처 레이블: 각 항목은 카메라, 갤러리, 또는 공유에서 가져온 것인지 표시합니다.

설정

다중 코드 모드
기본적으로 코드 리더는 코드 하나를 찾으면 바로 멈춥니다. 다중 코드 모드를 켜면 한 번에 최대 5개의 코드를 스캔할 수 있어, 명함이나 제품 시트처럼 여러 코드가 나란히 있는 이미지에 유용합니다.

다중 코드 모드가 활성화되면 카메라는 첫 감지 후 잠깐 더 스캔을 계속하여 결과를 표시하기 전에 모든 보이는 코드를 수집합니다. 그래서 단일 코드 모드보다 약간 더 시간이 걸립니다.

고급 세부 정보 표시
고급 세부 정보 표시를 켜면 각 스캔 결과 옆에 QR 버전, 오류 수정 레벨, 마스크 패턴, 바코드 구조(시작/종료 문자, 체크 숫자) 같은 기술적 메타데이터가 표시됩니다. 이 정보는 스캔 결과 페이지와 기록 탭의 행에 표시됩니다.

💡 QR 코드와 바코드의 실제 작동 방식에 대해 더 알아볼 수 있는 좋은 방법입니다. 스캔하는 기술이 궁금하다면 한번 사용해 보세요!

스캔 결과

코드가 감지되면 코드 리더는 아무 작업도 수행하기 전에 내용을 먼저 보여줍니다. 항상 원시 데이터를 먼저 보여주며 복사 및 공유 버튼이 함께 표시됩니다. 코드에 URL이 포함된 경우(QR 코드에서 일반적) 열기 버튼이 나타납니다. 이를 탭하면 앱 내 브라우저가 아닌 기본 브라우저에서 링크가 열립니다. 탐색 데이터는 수집되지 않습니다.

코드 리더는 특수 코드 유형을 자동으로 인식하고 상황별 작업을 표시합니다:

  • 디코딩된 데이터: 파싱 전에 바코드 안에 인코딩된 원시 문자열.
  • 구조: 바코드의 기술적 분석: 시작/종료 문자, 체크 숫자, 가드 패턴 및 코드 유형에 따른 유사 요소.

QR 코드의 경우 버전 번호, 오류 수정 레벨, 마스크 패턴을 보여주는 메타데이터 칩도 표시됩니다.

추적 매개변수 감지
URL에 추적 매개변수가 포함된 경우 코드 리더는 추적 없이 열기 버튼을 표시합니다. 여기에는 UTM 태그, Facebook, Google Ads, Microsoft, TikTok 등의 클릭 ID가 포함됩니다. 이를 탭하면 해당 매개변수가 제거된 상태로 링크가 열립니다. 추적 없이 동일한 페이지에 접근할 수 있습니다.

다른 앱에서 스캔하기

메시지, 이메일, 문서 뷰어, 웹 브라우저 등 다른 앱 내 사진에 나타나는 코드를 해당 앱을 떠나지 않고 스캔할 수 있습니다.

공유 사용:

  1. 갤러리 또는 앱에서 이미지를 열고 공유 버튼을 탭하세요.
  2. 나타나는 앱 목록에서 코드 리더를 찾아 탭하세요. 보이지 않으면 목록 끝까지 스크롤하여 더 보기를 탭하세요.
  3. 앱이 열리고 이미지를 자동으로 스캔합니다.

동작 사용:

  1. 갤러리 또는 앱에서 이미지를 열고 공유 버튼을 탭하세요.
  2. 앱 목록 아래의 동작 행을 스크롤하고 Scan for Codes를 탭하세요.
  3. 앱이 열리고 이미지를 자동으로 스캔합니다.
💡 두 옵션 모두 동일한 작업을 수행합니다. 앱 행의 "코드 리더"와 동작 행의 "Scan for Codes"는 동일한 기능에 접근하는 두 가지 방법입니다.

홈 화면 위젯

홈 화면에 앱으로 바로 이동하는 더 큰 단축키로 위젯을 추가할 수 있습니다. 탭하면 카메라가 바로 열려 스캔 준비가 됩니다. 위젯은 세 가지 크기(소, 중, 대)로 제공되므로 홈 화면 레이아웃에 맞는 크기를 선택할 수 있습니다.

  1. 아이콘이 흔들릴 때까지 홈 화면의 빈 영역을 길게 누르세요.
  2. 왼쪽 상단의 + 버튼을 탭하세요.
  3. 코드 리더를 검색하세요.
  4. 크기를 선택하고 위젯 추가를 탭하세요.
  5. 나중에 위젯을 길게 눌러 위젯 편집을 선택하면 크기를 조정할 수 있습니다.

Control Center

Control Center는 화면 오른쪽 상단 모서리에서 아래로 스와이프하면 나타나는 패널입니다. 잠금 화면이나 다른 앱을 사용하는 중에도 어디서든 한 번의 탭으로 카메라를 실행하는 코드 리더 버튼을 추가할 수 있습니다.

  1. 오른쪽 상단 모서리에서 아래로 스와이프하여 Control Center를 여세요.
  2. + 버튼을 탭하여 편집 모드로 진입하세요.
  3. 목록에서 코드 리더를 찾아 탭하여 추가하세요.
  4. 편집 모드에서 드래그하여 위치를 변경하고 크기를 조정할 수 있습니다.
